## 二、禁止使用的词汇（AI味高频词）

### 2.1 绝对禁止

| 词汇 | 替代方案 |
|------|---------|
| 瞳孔收缩 | 用具体动作（手攥紧、脚步停顿） |
| 嘴角微微上扬 | "他笑了" 或删除 |
| 不禁 | 删除 |
| 竟然 | 删除或改写 |
| 仿佛 | 删除或改写 |
| 宛如 | 删除或改写 |

### 2.2 谨慎使用（每章不超过1次）

| 词汇 | 使用场景 |
|------|---------|
| 猛然 | 真正的突然事件 |
| 突然 | 真正的突然事件 |
| 然而 | 必要的转折 |

## 三、禁止的写法

### 3.1 排比三连

❌ 禁止：
```
他看到了...
他听到了...
他感受到了...
```

✅ 改为：
```
远处的山峦在暮色中变成一道剪影，风从山谷吹来，带着松针的苦味。
```

### 3.2 空洞抒情

❌ 禁止：
- "命运的齿轮开始转动"
- "时间仿佛静止了"

✅ 改为：删除或替换为具体动作

### 3.3 情感标签

❌ 禁止：
- "他感到愤怒"
- "她很伤心"

✅ 改为：
- "他把杯子摔在地上"
- "她转过身，肩膀在抖"

## 四、推荐的写法

### 4.1 短句制造节奏

紧张时用短句，舒缓时用长句。

```
【紧张场景】
门开了。
黑暗。
一双眼睛。

【舒缓场景】
夕阳从西边的山峦后面沉下去，把整片天空染成了橘红色，像是有人在云层上泼了一桶颜料。
```

### 4.2 对话有个性

每个人说话方式不同。

```
【角色A】
"我觉得我们应该仔细分析一下当前的情况。"

【角色B】
"分析个屁，先干再说。"

【角色C】
"..."（沉默）
```

### 4.3 动作代替心理

"他攥紧拳头"比"他很愤怒"好一百倍。
