# 对话占比监控（v2.0.1新增）

> 基于大量实战审阅的经验总结
> 核心目标：防止全章变成聊天记录，确保叙事/氛围描写占比

---

## 一、实战教训

### 1.1 常见问题一览

| 场景 | 对话占比 | 评价 |
|------|:---:|------|
| 开篇章节 | ≤ 25% | ✅ 优秀，氛围为主 |
| 中段章节 | 26-35% | ✅ 良好，平衡得当 |
| 后期章节 | **> 50%** | 🔴 灾难性超标 |
| 高潮章节 | > 40% | 🔴 严重超标 |
| 决战章节 | > 35% | 🔴 偏高 |
| 收尾章节 | > 35% | ⚠️ 偏高 |
| 过渡章节 | > 40% | 🔴 超标 |

### 1.2 趋势分析

```
常见趋势：
  前期：对话占比 15-30%，健康
  中期：对话占比逐渐上升至 35-45%
  后期：对话占比普遍 40-60%，严重超标

结论：后期写作容易偷懒，用对话凑字数，叙事描写大幅减少
```

### 1.3 对话占比失控的危害

1. **氛围消失**：没有环境描写，读者无法沉浸
2. **信息注水**：大量无效对话，凑字数
3. **节奏拖沓**：对话推进慢，情节停滞
4. **AI味加重**：AI擅长写对话，不擅长写描写

## 二、对话占比标准

### 2.1 总体标准

| 对话占比 | 评级 | 说明 |
|:---:|:---:|------|
| ≤ 25% | ✅ 优秀 | 氛围/叙事为主，网文最佳 |
| 26-35% | ✅ 良好 | 平衡得当，可接受 |
| 36-40% | ⚠️ 偏高 | 需关注，下章注意控制 |
| 41-50% | 🔴 超标 | 必须压缩对话，补充叙事 |
| > 50% | 🔴 严重 | 章节需大改 |

### 2.2 按章节类型细分

| 章节类型 | 对话上限 | 说明 |
|---------|:---:|------|
| 战斗章 | 30% | 战斗描写为主 |
| 探索章 | 35% | 环境/氛围描写为主 |
| 悬疑章 | 25% | 氛围营造为主 |
| 过渡章 | 40% | 可适当放宽 |
| 对话章（剧情交代） | 45% | 特殊情况，需审查是否注水 |

### 2.3 按卷位置细分

| 卷位置 | 建议占比 | 说明 |
|-------|:---:|------|
| 开篇（前10章） | ≤ 25% | 建立氛围，吸引读者 |
| 中段 | ≤ 35% | 保持平衡 |
| 高潮 | ≤ 30% | 战斗/冲突为主 |
| 结尾 | ≤ 35% | 收束情节 |

## 三、对话占比计算方法

### 3.1 基础计算

```
步骤1：统计章节总行数（排除空行）
  total_lines = grep -c "." chapter.md

步骤2：统计对话行数（含引号的行）
  dialogue_lines = grep -c '"' chapter.md

步骤3：计算占比
  ratio = (dialogue_lines / total_lines) * 100%
```

### 3.2 精细化计算（可选）

```
对话字数统计：
  提取所有引号内内容
  统计字数
  dialogue_chars = 字数

章节总字数：
  total_chars = wc -c chapter.md / 3

占比：
  ratio = (dialogue_chars / total_chars) * 100%
```

### 3.3 快速估算

```
目测法：
  随机抽5个段落
  统计对话段落数
  估算占比

示例：
  5个段落中3个是对话 → 约60%对话占比
```

## 四、对话占比监控流程

### 4.1 每章监控

```
写完章节后：
  1. 计算对话占比
  2. 记录到 dialogue-monitor.md
  3. 判断是否超标
  4. 若超标，标记需修改
```

### 4.2 趋势监控

```
每5章回顾：
  1. 计算最近5章平均对话占比
  2. 检查趋势（上升/稳定/下降）
  3. 若连续3章上升 → 预警
```

### 4.3 全书监控

```
每卷结束：
  1. 统计本卷平均对话占比
  2. 对比其他卷
  3. 检查是否有"前低后高"崩塌趋势
```

## 五、对话占比超标处理

### 5.1 压缩对话

```
方法1：合并对话
  "XXX。"他说。
  "YYY。"她说。
  → "XXX。"他说，"YYY？"

方法2：删除无效对话
  - 重复确认的对话
  - 无信息量的寒暄
  - 过度解释的对话

方法3：转为叙述
  "他告诉了我事情的经过。"（代替大段对话）
```

### 5.2 补充叙事

```
方法1：增加环境描写
  - 光线、声音、气味
  - 温度、触感
  - 空间感

方法2：增加动作描写
  - 角色的细微动作
  - 表情变化
  - 视线移动

方法3：增加心理描写
  - 主角的思考过程
  - 情绪变化
  - 决策推理
```

### 5.3 修改示例

```
修改前（对话占比60%）：
  "我们该怎么办？"她问。
  "我也不知道。"他说。
  "要不要试试那个方法？"
  "哪个方法？"
  "就是之前说的那个。"
  "好像可以。"

修改后（对话占比30%）：
  她的声音有些发颤。他没有立刻回答，盯着墙上的裂缝看了三秒。
  "试试之前的方法。"他终于开口。
```

## 七、特殊情况处理

### 7.1 允许高对话占比的情况

- 审讯/拷问场景
- 关键信息揭露场景
- 情感爆发场景
- 谈判/辩论场景

即使这些情况，对话占比也不应超过50%

### 7.2 检查清单

- [ ] 对话是否推进了情节？
- [ ] 对话是否提供了新信息？
- [ ] 对话是否展示了角色性格？
- [ ] 是否有无效对话可删除？
- [ ] 是否可以转为叙述？

---

**记住：对话是手段，不是目的。过度依赖对话是写作偷懒的表现。**
