# 示例项目：《我能在诡异世界改写规则》

> 这是一个使用番茄Skill的示例项目，展示如何填充各个文件

---

## 项目概述

- **书名**：《我能在诡异世界改写规则》
- **题材**：规则怪谈 + 民俗悬疑
- **预估字数**：120万字
- **平台**：番茄小说

---

## 文件说明

本示例项目包含以下已填充的文件：

| 文件 | 内容 |
|------|------|
| `story_bible.md` | 世界观、怪谈域设定、解析眼规则 |
| `book_rules.md` | 平台规则、主角陆沉锁定、AI味禁忌 |
| `outline.md` | 四卷大纲、章节目标 |
| `current_state.md` | 第一章后的当前状态 |
| `pending_hooks.md` | 父亲失踪、老周生死等悬念 |
| `character_matrix.md` | 陆沉、父亲、赵铁生等角色设定 |
| `style_guide.md` | 风格指南、AI去味规则 |

---

## 使用方法

1. 复制 `project-template/` 作为你的项目起点
2. 参考 `examples/demo-novel/` 了解如何填充
3. 每章写前阅读核心文件
4. 每章写后执行检查
5. 更新状态文件
